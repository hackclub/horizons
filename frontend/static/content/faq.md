# FAQ

## What's a game jam?

A game jam is an event where you build a game from scratch. It's about teamwork, creativity, having fun, and learning new things.

## Who can participate?

This event is open to anyone 13-18, regardless of skill level.

## All this for free? Is this legit?

Yup, all for free! [Hack Club is a 501(c)3 non-profit organization](https://the.hackfoundation.org/), and we've run a bunch of events like this in the past. For example, we ran a [180-person hardware hackathon at GitHub HQ](https://www.youtube.com/watch?v=kaEFv7e49mo), and a [seven-day game jam in Shanghai, China](https://www.youtube.com/watch?v=Gtjyyu82pw4).

## What if my parents are concerned?

We're here to help! Feel free to reach out to [flagship@campfire.hackclub.com](mailto:flagship@campfire.hackclub.com) and we'd love to answer questions or hop on a call.

## What is Hack Club's child protection policy?

Hack Club takes the safety and well-being of all participants seriously. You can read our full [safeguarding policy](https://hackclub.com/safeguarding-policy) to learn about our commitment to child protection.

## I'm not good at coding / I've never made games before. Can I still participate?

Yes, we want you to! This game jam is for all skill levels. We'll be running workshops and support calls to help everyone learn together.

## What can I make?

Any kind of game is allowed–platformers, puzzlers, visual novels, clicker games–anything!

## Can I use AI?

Use of generative AI **for graphics and audio** is **prohibited**. Use of generative AI for other purposes is allowed, as long as less than 25% of the code in your game was written with the assistance of generative AI. Additionally, you shouldn't write updates or game descriptions using AI.

## Does making art or music count as game development?

Yes; artists and musicians are encouraged to participate even if they've never coded before. However, this is a technical event, so a maximum of ⅓ of the total logged time can be spent on music or art.

## Can I work in a team?

Yes! Teamwork is heavily encouraged. You must log your work separately from your teammates, and art or music can make up a maximum of ⅓ of the time spent on the game.

## How many games can I make while qualifying?

You can make as many as you want! Games will only count towards your progress when they're in a playable state– they don't need to be finished, but anything you make should be able to be called a 'demo' by the end of the event.

## Can I submit an existing project?

Existing projects are allowed, but you can only submit time that was logged after the start date.

## Does my project need to be on GitHub?

All projects must be open source on GitHub or another similar software forge.

## What languages or game engines can I use? Is Scratch allowed?

Any **code** game engines are allowed. This means that Scratch (including TurboWarp) is not allowed, because it's block coding. Additionally, GDevelop isn't allowed because of the level of AI usage.

## Can I submit to this event and another Hack Club program?

Sorry, nope! "Double-dipping" is not allowed. If you make a game for this event, you can only submit it here, unless another program specifically states otherwise.

## What's the minimum complexity or amount of content for a game?

We understand that making games is hard, and that it might not be easy for everyone to make a finished game. However, to count as a playable demo, "finished" games should have at least five minutes of engaging gameplay.

## I'm 18, but I'm going to turn 19 by the end of this event. Can I still participate?

You can make games, join AMAs, and win prizes, but you can only attend the in-person event if you're 18 or under at the time of the event.

## Can I get involved or contribute?

We love this question! We'd love more help running workshops, support calls, or even help organizing AMAs with real game developers. If any of these interest you, let us know!
