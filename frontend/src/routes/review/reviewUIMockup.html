CatCapes

Change your in-game capes, visible to others using this mod.

Installation

Download from Modrinth and place in your Fabric mods folder. Requires Minecraft Java Edition 1.21.x.

How to Use

Online mode: Log on to the catcapes.jgj52.hu Minecraft server, type /token and click the message to copy it. Go to ModMenu and set the token setting to what you just copied.

Offline mode: In the config turn off online mode. Then set your cape as usual.

Custom Textures

Go to catcapes.jgj52.hu and upload your cape texture. You can make these textures using something like skinmc.net/capes/editor.
